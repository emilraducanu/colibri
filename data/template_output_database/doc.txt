The colibri pipeline should produce two file as the output:
    - 'MA.json' for meta-analyses
    - 'PS.json' for primary studies

These two databases are linked, i.e. for each entry of 'MA.json' you can find a list of primary
studies which the meta-analysis is based on. These primary studies are stored into 'PS.json'.

Here is a description of what you can find in those two files.
In column 'Variable', the name of the variable is stored.
In column 'Type', the variable type is stored (int, str, float, list, bool).
In column 'Possible values', an exhaustive list of possible values of the variable is stored
when available. If nothing is specified, it means the variable can take any value (based on type).
Note that the 'DOI' variable acts as the unique ID for each entries of 'MA.json' and 'PS.json'.
Note that any variable, exept 'DOI', can contains N/A.
Note that the possible values of variable 'Country studied' in 'PS.json' are based on standard ISO 3166-1 alpha-2.

In addition, you can find examples of entries in 'MA_examples.json' and 'PS_examples.json'.
